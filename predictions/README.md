# 预测结果记录系统

本目录用于保存每次出方案时的预测记录和后续结果跟踪，便于长期评估 skill 的真实命中率。

---

## 文件说明

| 文件 | 用途 |
|:---|:---|
| `README.md` | 本说明文件 |
| `prediction-template.md` | 单次预测记录模板 |
| `results-tracking.md` | 累计结果跟踪表 |
| `YYYY-MM-DD-prediction.md` | 每次出方案后自动生成的记录 |

---

## 记录流程

1. 每次出方案后，复制 `prediction-template.md` 为 `YYYY-MM-DD-prediction.md`
2. 填写预测内容、赔率、投注方向、信心等级
3. 比赛结束后，回填实际结果和盈亏
4. 定期更新 `results-tracking.md` 中的累计统计

---

## 记录原则

- **诚实记录**：无论输赢都要记录，不能只记盈利
- **及时回填**：比赛结束后尽快回填结果
- **分类统计**：按规则类型、联赛、日期等维度统计
- **保护隐私**：如果涉及真实金额，可以只记倍数/比例，不记绝对金额
