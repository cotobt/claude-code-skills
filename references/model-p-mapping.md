# 模型p（模型概率）确定方法

## 核心原则

模型p = 市场p + 评分净分映射加成

**不要**单独从泊松λ修正推导模型p用于凯利计算。泊松λ只在Step 8作为比分概率的交叉验证参考。

## 评分净分映射表

| 评分净差范围 | 模型p加成 | 说明 |
|:-----------:|:---------:|------|
| 0~5 | 市场p + 1~2% | 微弱优势，谨慎 |
| 6~10 | 市场p + 3~5% | 中等优势 |
| 11~15 | 市场p + 5~10% | 明显优势；当赔率<1.50时取上限8~10%(低赔需要更高模型p才能通过>1.0检查) |
| 16~20 | 市场p + 8~12% | 显著优势 |
| 21~25 | 市场p + 12~15% | 碾压（注意让球盘口径差异） |
| 26+ | 市场p + 15%+ | 碾压级 |

## 绝对检查

模型p × 赔率 > 1.0

如果 < 1.0：
1. 检查评分是否算错（净分是否过低？维度是否遗漏？）
2. 如果确认评分无误，仍按模型p出方案
3. 凯利f≤0时，金额取最小值¥2

## 与旧法（泊松λ修正）的关系

旧法：λ从赔率反推 ± 评分修正（每±3分→λ±0.1）→ 泊松 → 模型p

旧法问题：λ设定值主观性大，容易算出不合理的低概率。例如美国λ=1.96、澳大利亚=0.73，泊松打出美国胜率仅68%，明显偏低。

新法：直接映射，更稳定。泊松仅在Step 8用于比分概率参考，不用于凯利计算。
