# 🚌 连乘链计算示例

## 案例：巴西 vs 海地（2026-06-20）

### 基础触发

| 条件 | 结果 |
|:-----|:----:|
| 巴西传控（首轮62%控球+15射门） | ✅ |
| 海地大巴+FIFA排名差80≥20 | ✅ |
| 热门非东道主 | ✅ |
| 排名差80≥20 | ✅ |
| **触发基础分** | **3** |

### 连乘修正

| 修正项 | 数据来源 | 系数 |
|:-------|:---------|:----:|
| 传控中效（首轮射正5次，预期进球约1.2） | 首轮vs摩洛哥：62%控球15射门5射正1进球 | ×0.7 |
| 无精英高中锋（维尼修斯/罗德里戈非高中锋） | 首发无195cm+中锋 | ×0.9 |
| 缺组织核（进攻三区传控效率一般） | 首轮15射门仅1进球，缺乏中场创造者 | ×0.8 |
| 赔率1.73>1.40 | 不忽略 | 1.0 |

**最终=3×0.7×0.9×0.8=1.51** → 0-1区间→无冷门

### 解读

- 巴西vs排名83的海地，实力碾压级别
- 传控质量中效+缺高中锋+缺组织核，破大巴能力受限
- 但1.51的🚌分表示"几乎没有冷门价值"，选热门方向（巴西让胜）
- 首轮1-1平局反而提升战意，不会轻敌
