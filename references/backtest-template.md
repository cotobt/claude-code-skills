# 回测模式输出模板

本文件定义 skill 在回测模式下的标准化输出格式。

---

## 一、触发条件

当用户明确说以下关键词时，进入回测模式：
- "回测"
- "复盘"
- "验证"
- "历史"
- "回测今天结束的"
- "回测 [日期] 的比赛"

回测模式下：
- 数据源可以来自多个渠道（参见 `references/data-sources.md`）
- 不需要出投注方案
- 重点验证 skill 核心规则的有效性

---

## 二、输入要求

进行回测前，必须确认：
1. 回测日期范围
2. 比赛结果（对阵 + 比分）
3. 赛前赔率（如有）
4. 关键统计数据（控球率、射门、射正、xG 等，如有）

如果用户未提供完整数据，应明确说明缺失项，并先用可用数据做定性回测。

**自动化工具**：可使用 `scripts/backtest.py` 根据赛果 JSON 自动生成回测报告框架：

```bash
python scripts/backtest.py --results scripts/backtest-example.json --output backtest-report.md
```

然后人工补充规则触发情况和关键案例分析。

---

## 三、标准化输出结构

### 1. 回测概览表

| 项目 | 数值 |
|:---|:---|
| 回测日期范围 | YYYY-MM-DD 至 YYYY-MM-DD |
| 总场次 | N |
| 有赔率场次 | N |
| 无赔率场次 | N |
| 数据来源 | 500.com / FBref / Understat / 用户提供 等 |

---

### 2. 场次清单与规则触发表

| 日期 | 比赛 | 实际结果 | 赛前热门 | 触发规则 | 规则方向 | 是否命中 | 备注 |
|:---|:---|:---:|:---|:---|:---:|:---:|:---|
| 示例：6/20 | 土耳其 vs 巴拉圭 | 0-1 | 土耳其 | 🎈伪传控 + 🚌情景大巴 | 巴拉圭不败 | ✅ | 土耳其78%控球0进球 |

**触发规则代码：**
- 🚌：冷门检测（传控vs大巴）
- 🎈：伪传控陷阱
- 📉：对手质量折扣
- 📈：排名差+首轮交叉校准
- 🏠：东道主加成
- 🐴：黑马球队标记
- 🔒：防线漏洞检查

---

### 3. 规则触发统计表

| 规则 | 触发次数 | 命中次数 | 命中率 | 备注 |
|:---|:---:|:---:|:---:|:---|
| 🚌深防（完整） | 0 | 0 | — | |
| 🚌深防（降权） | 0 | 0 | — | |
| 🚌深防（低效） | 0 | 0 | — | |
| 🎈伪传控（单轮） | 0 | 0 | — | |
| 🎈伪传控（跨轮） | 0 | 0 | — | |
| 📉对手质量折扣 | 0 | 0 | — | |
| 📈排名差+首轮校准 | 0 | 0 | — | |
| 🏠东道主加成 | 0 | 0 | — | |
| 🐴黑马球队标记 | 0 | 0 | — | |
| 🔒防线漏洞检查 | 0 | 0 | — | |

---

### 4. 方向判断命中率表

| 判断类型 | 场次 | 命中 | 命中率 |
|:---|:---:|:---:|:---:|
| 热门方正路打出 | 0 | 0 | — |
| 冷门/下盘打出 | 0 | 0 | — |
| 让球盘命中 | 0 | 0 | — |
| 标准SPF命中 | 0 | 0 | — |

---

### 5. 关键案例分析

选择 2-4 场最具代表性的比赛深入分析：

#### 案例X：[比赛名称]
- **赛前背景**：球队排名、近期状态、战意
- **触发规则**：哪些规则被触发
- **赛前判断**：根据规则应该押什么方向
- **实际结果**：比分和打出选项
- **归因**：为什么对/为什么错

---

### 6. 经验教训表

| 序号 | 发现 | 对应规则 | 后续优化 |
|:---:|:---|:---|:---|
| 1 | 示例：高控球低射正球队容易翻车 | 🎈伪传控 | 强化首轮低效率预警 |
| 2 | | | |

---

### 7. 局限性说明

每次回测必须说明：
- 哪些数据缺失（如赔率、xG、控球率）
- 判断是基于实际赔率还是事后推断
- 样本量是否足够
- 是否存在幸存者偏差

---

## 四、禁止事项

- ❌ 不要编造赔率或统计数据
- ❌ 不要用结果反推规则触发（规则必须在赛前可判断）
- ❌ 不要夸大命中率（必须说明样本量和局限性）
- ❌ 不要只做简单比分罗列，必须做规则验证

---

## 五、示例片段

```markdown
## 回测概览
回测 2026-06-12 至 2026-06-22 共 44 场比赛。

## 规则触发统计
| 规则 | 触发 | 命中 | 命中率 |
|:---|:---:|:---:|:---:|
| 🚌深防 | 6 | 6 | 100% |
| 🎈伪传控 | 3 | 3 | 100% |

## 关键案例
**土耳其 0-1 巴拉圭**：土耳其78%控球但0进球，🎈伪传控触发，🚌情景大巴触发，让负命中。

## 经验教训
高控球低射正球队需要降低评分，不能只看控球率判断优势。
```
