# 竞彩知识参考

本文件汇总了竞彩投注策略文章中提炼的核心知识点，供出方案时参考。

## 一、价值投注理论（Value Betting）

来源：知乎《足球竞彩的科学分析方法与策略优化》

**核心原则：**
- 可持续盈利建立在"价值投注"基础上
- 模型真实概率 > 庄家隐含概率时才有正期望
- 不追求每场命中，追求长期期望值管理

**隐含概率计算：**
- 单场隐含概率 = 1÷赔率
- 归一化真实概率 = 单场隐含概率 ÷ 总概率和（去庄家抽水）
- 多出来的部分（约9%）是庄家利润

**价值判定：**
- 模型p - 市场p ≥ 5% → 价值投注空间，可出手
- 价值空间 < 5% → 不投

## 二、串关概率与策略

**概率规律：**
- 单场胜平负命中率 ≈ 33%（纯随机）
- 2串1命中率 ≈ 11%（33%×33%）
- 3串1命中率 ≈ 3.6%
- 4串1以上不足1%，不建议

**最优策略：**
- 2串1：收益与风险平衡最佳
- 3串2（3个2串1容错）：兼顾收益与容错
- 避开4串及以上高串关

## 三、泊松分布应用

**计算步骤：**
1. 确定主客队预期进球 λ1、λ2（从赔率反推）
2. 枚举所有符合条件比分组合
3. 对比分逐个计算概率：P(k) = e^(-λ) × λ^k / k!
4. 求和得到总概率

**示例：** λ1=1.2, λ2=0.8时，小球(≤2.5)概率≈67.56%

## 四、资金管理

**纪律：**
- 单月投入 ≤ 月收入15-20%
- 单注 ≤ 总资金5%
- 当日亏损达本金20%时停止
- 连续3场亏损后强制休息48小时
- 建立投注日志记录每笔决策

**凯利公式：**
- f = (p×(b+1)-1) / b
- p=胜率, b=赔率-1
- 建议采用半凯利（50%理论值）降低波动

## 五、联赛选择

**优先级：**
1. 五大联赛（英超/西甲/意甲/德甲/法甲）— 数据透明、监管严格
2. 世界杯/欧洲杯等大赛 — 关注度高、信息充分
3. 避开不知名野赛 — 假球风险高、信息不对称

**80/20原则：** 80%资金集中3-5个最熟悉的联赛

## 六、盘口解读

**异常信号：**
- 实力差距与让盘严重不匹配
- 临场赔率非基本面驱动的剧烈波动
- 不同机构开出显著分歧赔率
- 及早投注：最早盘口最准确，临场可能调赔

**高低赔误区：**
- 低赔(1.2-1.6)容易出下盘 — 警惕"庄家忽悠"
- 高赔冷门(20倍+)概率极低 — 长期必亏
- 反大众心理：不能只从排名和实力看

## 七、心态与习惯

- 每天只分析2-3场，避免盲目投注
- 有赚就好，低赔也是肉（1.2×1.2=1.44收益44%）
- 拒绝倍投（2→4→8→16模式），一次失败全亏
- 竞彩本质是娱乐，返奖率仅71-73%，长期负预期
